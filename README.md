<h1>To-Do List CLI</h1>



<p><b>A simple command-line to-do list application written in Python.</b></p>



<h2>Description</h2>

<p>This project lets you view, add, and remove items from a to-do list directly in your terminal.

Tasks are saved to a local <code>todos.txt</code> file so they persist between runs. This repository

was created as part of a university Git \&amp; GitHub assignment to practice commits, branching, and

merging.</p>



<h2>Features</h2>

<ul>

&#x20; <li>View current to-do items</li>

&#x20; <li>Add new items</li>

&#x20; <li>Remove items by number</li>

&#x20; <li>Mark items as high priority</li>

&#x20; <li>Persistent storage using a text file</li>

</ul>
